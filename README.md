# Calculator
calculator by Reverse Polish Notation.

## How to make
`make`

## Usage
In terminal, execute calc file.  
`./calc`  

### Example input
```
10 + 5
10+5
3*(4+6)-7
sin(45+30) + cos45 + tan(90 / (4 - 2))
10.5*sin(2.5*100)
$abc = 20 * 3
$abc / 4
2 * $prev + 10
```

### Exit
Enter `quit` or `quit()`

## Support

* Four arithmetic operations  
* Brackets  
* Trigonometric functions  
* Variable
